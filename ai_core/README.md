# AI Core

Интеллектуальный модуль оценки действий оператора, классификации ошибок и прогнозирования аварийных рисков в реальном времени.

## Основные задачи
1. Классификация последовательностей действий обучаемого и сравнение с эталонными сценариями.
2. Прогнозирование вероятности аварии на основе текущих технологических параметров (давление, температура, уровень в колонне).
3. Генерация адаптивных сценариев повторного обучения и подсказок.

## Планируемый стек
* Python
* PyTorch / Scikit-learn
* Pandas, NumPy

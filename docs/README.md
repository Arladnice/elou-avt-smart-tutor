# Документация проекта

В этой папке хранятся текстовые и аналитические материалы по проекту КТК ЭЛОУ-АВТ.

## Структура папки
* `/security` — Анализ ИБ-угроз, политика безопасности, ролевой доступ и защита моделей ИИ.
* `/economy` — Расчет экономической эффективности тренажера, окупаемость проекта и коммерческий потенциал.
* `/architecture` — Схемы интеграции компонентов, API спецификации и диаграммы процессов.

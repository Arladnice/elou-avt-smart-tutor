# 🏛 Документация проекта КТК ЭЛОУ-АВТ Smart Tutor

В данном каталоге собраны отчетные, проектные и справочные материалы по разработке Компьютерного тренажерного комплекса (КТК) первичной переработки нефти ЭЛОУ-АВТ в соответствии с 8 критериями оценки кейса.

---

## 📂 Структура каталога

### 📄 Основные проектные документы (Разделы для сдачи)
*   **[main_strategic_plan.md](file:///e:/Git-Projects/elou-avt-smart-tutor/docs/main_strategic_plan.md)** — Главный стратегический план команды. Содержит GAP-анализ по 8 критериям оценки, план спринтов и чек-лист готовности проекта.
*   **[solution_architecture.md](file:///e:/Git-Projects/elou-avt-smart-tutor/docs/solution_architecture.md)** — Архитектура IT-решения (по модели 4+1, диаграммы интеграции, процесса, логики и развертывания).
*   **[ai_architecture.md](file:///e:/Git-Projects/elou-avt-smart-tutor/docs/ai_architecture.md)** — Архитектура ИИ-модуля (детализация LSTM-прогноза рисков и DTW-анализа ошибок).
*   **[infrastructure.md](file:///e:/Git-Projects/elou-avt-smart-tutor/docs/infrastructure.md)** — Инфраструктурное решение (аппаратные требования, резервирование, latency-контроль, мониторинг Prometheus+Grafana).
*   **[requirements.md](file:///e:/Git-Projects/elou-avt-smart-tutor/docs/requirements.md)** — Техническое задание (БТ, ФТТ, НФТ по ГОСТ 34.602-89).
*   **[economics.md](file:///e:/Git-Projects/elou-avt-smart-tutor/docs/economics.md)** — Финансово-экономическое обоснование (TCO, NPV, PI, IRR, DPP, дерево КПЭ).
*   **[market_analysis.md](file:///e:/Git-Projects/elou-avt-smart-tutor/docs/market_analysis.md)** — Анализ рынка, сравнение существующих OTS-решений и конкурентные преимущества.
*   **[legal_compliance.md](file:///e:/Git-Projects/elou-avt-smart-tutor/docs/legal_compliance.md)** — Регуляторное соответствие (Ростехнадзор, ГОСТ, ФЗ-152) и лицензионная чистота.
*   **[security_threat_model.md](file:///e:/Git-Projects/elou-avt-smart-tutor/docs/security_threat_model.md)** — Модель угроз ИБ (RBAC, криптографический аудит результатов SHA-256).

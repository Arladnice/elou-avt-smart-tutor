# Simulator ЭЛОУ-АВТ

Математические и физические модели процессов переработки нефти установки ЭЛОУ-АВТ (атмосферная и вакуумная перегонка, электрообессоливание).

## Задачи симулятора
1. Расчет материальных и тепловых балансов процессов в реальном времени.
2. Имитация изменения параметров (давление, температура, расход, состав сырья) в ответ на действия оператора КТК.
3. Моделирование нештатных ситуаций и отказов оборудования.
